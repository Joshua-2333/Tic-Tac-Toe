## Overview

Tic-Tac-Toe is a simple two-player game played on a 3×3 grid where players take turns placing their symbols (X or O). The goal is to be the first to align three symbols horizontally, vertically, or diagonally. :contentReference[oaicite:0]{index=0}  

This project recreates the game in the browser and demonstrates:

- Game state management  
- Turn-based logic  
- Conditional win/draw detection  
- Dynamic DOM updates  

---

## How to Play

1. Player 1 is **X**, Player 2 is **O**  
2. Players take turns selecting empty cells  
3. First to get **3 in a row** wins  
4. If all cells are filled with no winner → it's a draw
